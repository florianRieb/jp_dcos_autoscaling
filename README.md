# jp_dcos_autoscaling

Auto-Scaling Komponente für Marathon Apps.

Mit Hilfe der Parameter welche über Args übergeben werden, werden Schwellenwerte für die CPU und Memory Auslatung festgelegt,
anhand der die Container horizontal skaliert werden.

Die aktuelle Version dient nur zum testen für die Open DC/OS V. 1.9 und ist eine angepasste Version von:

https://github.com/mesosphere/marathon-autoscale
